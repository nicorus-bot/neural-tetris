# Neural Tetris VS CPU

A modern, cyberpunk-themed Tetris battle game against a heuristic-based AI.

## Features
- **Cyberpunk UI**: Glassmorphism and neon effects.
- **CPU Battle**: Fight against an AI that simulates the best moves.
- **Battle Effects**: Screen shake, flashes, and line-clear animations.
- **Classic BGM**: Includes the iconic Tetris theme.
- **Responsive**: Auto-scaling cellSize.

## How to Play
- **Arrows**: Move and Rotate
- **Space**: Hard Drop
- **C**: Hold Piece

## Setup
```bash
npm install
npm run dev
```
